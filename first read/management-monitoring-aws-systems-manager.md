# AWS systems manager
---
- **key insights:**  ^a0aa4b
	- centralize operational data from multiple AWS services and automate tasks across your AWS resources
- **features** 
	- Create logical groups of resources: 
		- applications, different layers of an app stack, or production/development environments
	- select a resource group and view its recent API activity, resource configuration changes, related notifications, operational alerts, software inventory, and patch compliance status
	- Collects info on your instances and software installed on them.
	- Helps ensure that your software is up-to-date and meets your compliance policies
	- schedule time to run admin/maintenance tasks across your instances
- **pricing** 
	- For your own packages, pay for what you use
	- Package transfer to Distributor: 
		- charge based on size/storage duration, 
		- number of Get/Describe API calls, 
		- data transfer out-of-region/on-premises.
	- charged on number of :
		- Automation steps + type.Â 
		- OpsItems, change requests, API requests.
		- OpsItems created and runbook steps executed.
		- configuration requests and received.
		- advanced parameters stored and instances activated.
